# 🛒 Shopping Cart Data Wrangling

A Python project for performing data wrangling on a Shopping Cart dataset. This project demonstrates the complete data preprocessing workflow including assessing data quality, cleaning datasets, handling missing values, removing duplicates, correcting data types, and exporting clean datasets for further analysis.

---

## 📌 Project Overview

Raw datasets often contain inconsistent formats, duplicate records, missing values, and incorrect data types.

This project focuses on transforming raw shopping cart data into a clean and analysis-ready dataset using Python and Pandas.

---

## 📂 Dataset

The project consists of four datasets:

- Customer
- Orders
- Products
- Sales

---

## 🛠️ Data Wrangling Process

The notebook covers:

- Assessing data quality
- Checking data types
- Identifying missing values
- Detecting duplicate records
- Removing duplicates
- Handling null values
- Converting data types
- Standardizing data
- Exporting cleaned datasets
